# Alura Store Sales Analysis

## Descripción del Proyecto
Este proyecto analiza el desempeño de ventas de cuatro tiendas de la cadena ficticia **Alura Store**.  
El objetivo del análisis es evaluar diferentes métricas comerciales para identificar cuál tienda presenta el menor rendimiento y recomendar cuál debería vender el Sr. Juan para iniciar un nuevo emprendimiento.

El análisis incluye exploración de datos, cálculo de métricas de ventas y visualización de información para comprender el comportamiento de las tiendas.

---

## Objetivos del Análisis
- Explorar los datasets de ventas de las cuatro tiendas.
- Calcular el ingreso total generado por cada tienda.
- Analizar las categorías de productos más vendidas.
- Evaluar la valoración promedio de los clientes.
- Identificar los productos más vendidos y los menos vendidos.
- Calcular el costo promedio de envío por tienda.
- Generar visualizaciones para interpretar los resultados.

---

## Tecnologías Utilizadas
- Python
- Pandas
- Matplotlib
- Jupyter Notebook / Google Colab

---

## Dataset
Los datasets contienen información sobre las ventas realizadas en cada tienda de **Alura Store**.

El conjunto de datos incluye variables como:

- Producto vendido
- Categoría del producto
- Precio de venta
- Costo de envío
- Fecha de compra
- Valoración del cliente
- Método de pago
- Ubicación geográfica de la compra

---

## Proceso de Análisis
El flujo de trabajo realizado en el notebook incluye:

1. Carga de los datasets de las cuatro tiendas.
2. Exploración inicial de los datos.
3. Cálculo de ingresos totales por tienda.
4. Análisis de ventas por categoría.
5. Cálculo de la valoración promedio de clientes.
6. Identificación de productos más y menos vendidos.
7. Cálculo del costo promedio de envío por tienda.
8. Generación de gráficos para visualizar los resultados.
9. Elaboración de una recomendación final basada en los datos.
